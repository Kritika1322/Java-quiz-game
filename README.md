# Java-quiz-game 

An interactive command-line quiz game built in Java.

## Features
- 10 multiple choice questions
- A/B/C/D answer input
- Case insensitive answers (a or A both work)
- Score tracking
- Grade calculation (A+, A, B, C, F)
- Personalised messages based on performance

## Concepts used
- Variables and data types
- Arrays
- For loops
- If/else statements
- Methods
- Scanner (user input)
- String methods (.toUpperCase(), .equals())

## How to run
1. Make sure Java is installed
2. Compile: javac QuizGame.java
3. Run: java QuizGame
